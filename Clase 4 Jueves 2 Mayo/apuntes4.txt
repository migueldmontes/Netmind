Combinacion de selectores:

div p {  (A todos los hijos)
color: green;
}

div < p { (Solo a los hijos directos)
background: lightblue;
}

div span p { (Solo afecta a span)
border-right: 5px solid green;
}

div + p { (Al siguiente hermano de div)
font-weight: bold;
}

h3 ~ p { (Todos los siguientes hermanos)
border-right: 5px solid blue;
}

p [title] {
background color: lime;
}

.texto {
    font-family: Arial;
    color: red;
}

li.texto
color: green;

* = Cualquier cosa
